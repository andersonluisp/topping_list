# Aplicativo Lista de Coberturas
Aplicativo desenvolvido para exibir a lista de coberturas retornadas por um JSON.

## Tecnologias e Ferramentas utilizadas
- Arquitura MVVM
- LiveData
- Consumo de JSON através do Retrofit
- Coroutines
- Layouts em XML
- Fragments
- RecyclerView
